## Run Locally

Clone the project

```bash
  git clone https://github.com/ibnerasheed/dsa-maktab.git
```

Go to the project directory

```bash
  cd dsa-maktab
```

Set Envirionment Variables

```bash
  export PYTHONPATH=./src
```
